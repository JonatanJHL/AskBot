# AskBot - Asistente de IA para tu Empresa

Pregunta a tu base de datos en lenguaje natural.

## Instalación

### Trial (7 días gratis)

```bash
cd trial
php install.php
```

### Versión Completa

```bash
cd full
php install.php
```

### Web Widget

```bash
cd web
php install.php
```

## Características

- Consulta tu base de datos en lenguaje natural
- Multi-canal: Web, WhatsApp, Telegram
- Escalamiento a humano cuando no puede responder
- Adaptación automática a tu estructura de BD
- Sistema de licencias con trial
